Wagyz — Modélisateur de stratégie de migration
v17 · Import Silae obligatoire · Analyse combinée par dossier · Charge réelle · Vagues cohérentes · Export Excel
En attente d'import
Ce guide décrit les extractions à réaliser dans Silae avant d'utiliser le modélisateur. L'ensemble du travail de diagnostic repose sur ces 3 fichiers.
1
Extraction "Gestion interne simplifiée" — Silae Back-office

Dans Silae, accéder au menu Reporting → Gestion interne → Export simplifié. Cocher les colonnes : Numéro dossier, Raison sociale, Mode de gestion (Externalisation / Mise à disposition), Nombre de salariés, Coffre-fort actif.

Ce fichier correspond à l'Analyse des dossiers (colonnes G = mode, K = salariés, M = coffre-fort, AA→ = CCN).

✅ Fichier 1 : Analyse_des_dossiers.xlsx
2
Extraction MySilae Entreprise — fonctionnalités activées par dossier

Dans Silae, accéder à MySilae → Administration → Synthèse des accès entreprise. Exporter la liste avec les fonctionnalités actives par dossier. La colonne F "Absences et mobilité" (marquée X) indique la gestion des congés payés sur Silae — critère bloquant pour la migration Wagyz.

✅ Fichier 2 : Fonctionnalité_Silae.xlsx
3
Table de référence CCN Wagyz / Silae

Ce fichier contient la correspondance entre les codes CCN Silae (format lettre+3 chiffres) et les conventions collectives disponibles sur Wagyz. Il est fourni par l'équipe Wagyz et mis à jour à chaque nouvelle CCN paramétrée. Il sert à déterminer si chaque dossier est migrable.

Statuts possibles : En production (migrable), A recetter (migrable sous conditions, option à activer), Recette bloquée / Non référencée (non migrable).

✅ Fichier 3 : ccn_wagyz_silae.xlsx ℹ Table embarquée dans l'outil (91 CCN)
4
Ce que le modélisateur fait automatiquement à partir de ces fichiers

• Correspondance CCN : RECHERCHEV automatique entre codes CCN Silae et table Wagyz via le code IDCC
• Tableau croisé : mode de gestion × disponibilité CCN × CP Silae → statut migrable ou non par dossier
• Attribution des vagues : chaque dossier reçoit une vague en fonction de la combinaison de ses critères
• Calcul de charge : Σ(60 min + nb_salariés × 3 min) par dossier réel, pas de moyennes fictives
• Export Excel : liste complète avec vague, raison de blocage, coffre-fort, CP Silae
